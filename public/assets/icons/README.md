# Ícones do PWA

Esta pasta contém todos os ícones necessários para o PWA (Progressive Web App).

## Ícones Necessários

Para funcionalidade completa, você precisa fornecer os seguintes ícones:

### Favicons (Aba do Navegador)
- `favicon-16x16.png` - 16x16 pixels
- `favicon-32x32.png` - 32x32 pixels
- `../favicon.ico` - Multi-size ICO file

### PWA Icons (App Instalável)
- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels ✅ (placeholder instalado)

### Apple Touch Icon (iOS)
- `apple-touch-icon.png` - 180x180 pixels

### Open Graph (Redes Sociais)
- `og-image.png` - 1200x630 pixels (para WhatsApp, Facebook, etc.)

## Como Substituir

1. Siga o guia completo em `COMO_ADICIONAR_LOGO.md` na raiz do projeto
2. Use ferramentas online como:
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator)
3. Substitua os arquivos nesta pasta pelos seus ícones personalizados
4. Mantenha os mesmos nomes de arquivo

## Importante

⚠️ **Os ícones placeholder atuais são apenas para teste!**

Para melhor aparência:
- Use imagens PNG com fundo transparente ou sólido
- Mantenha o design simples e reconhecível em tamanhos pequenos
- Use alta qualidade/resolução
- Teste em diferentes dispositivos

---

✨ **Dica**: Crie primeiro um ícone de 512x512px e use as ferramentas online para gerar automaticamente todos os tamanhos necessários!
